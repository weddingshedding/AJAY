# Ajay Saini Portfolio

A premium, responsive portfolio for Ajay Saini, focused on website development, video editing, AI creative work and app experiences.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Firebase Hosting ready

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
```

The static website is generated inside the `out` folder.

## Firebase Hosting

1. Copy `.firebaserc.example` to `.firebaserc`.
2. Replace `YOUR_FIREBASE_PROJECT_ID` with your Firebase project ID.
3. Run `npm run build`.
4. Run `firebase deploy --only hosting`.
